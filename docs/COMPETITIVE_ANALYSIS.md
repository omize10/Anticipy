# Anticipy — Competitive Market Analysis (April 2026)

## Market: $43.64B (2025) → $310B+ (2033), 27.8% CAGR

## Competitors

### Humane AI Pin — DEAD
HP acquired $116M Feb 2025. Only 10K shipped, returns > sales. Lesson: high price + bad execution = death.

### Rabbit R1 — Recovering  
$199, 50K+ sold. RabbitOS 2 improving. Lesson: iterate + low price = survival.

### Meta Ray-Ban — Market Leader
$799 glasses, 7M+ units 2025 (3x YoY), 40%+ share. Anticipy is orthogonal (audio vs visual).

### Plaud NotePin — Active (Niche)
HIPAA transcription for professionals. No autonomous execution. Anticipy adds action.

### Limitless Pendant — DEAD
Meta acquired Dec 2025. Was $99, ~5K units. Lesson: indie players get acquired.

### Friend (Avi Schiffmann) — Backlash
$129, Museum of Failure. Always-listening + no value = reputational disaster.

### Bee AI (Amazon) — Closest Competitor
$49.99 + $19/mo. Amazon acquired Jan 2026. Does email/calendar. Anticipy does ANY web task via browser agent.

## Anticipy Differentiation
1. Autonomous browser execution (any web task, not just transcribe)
2. Under $100 hardware ($28 CAD BOM)
3. Privacy-first (on-device VAD, local processing)
4. Learns from competitors' failures

## Recommendations
1. Position: "Listens. Understands. Does."
2. Start B2B before consumer
3. Price $149-$299 consumer
4. Publish privacy policy before launch
